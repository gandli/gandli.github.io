## 🛠️ 技能大扩军

从 ClawHub 和 skills.sh 两个生态大量安装技能：

- **ClawHub**：编程、UI/UX、清洁代码审查等
- **skills.sh**（npx skills）：发现了更大的生态，6万+安装量
  - Vercel 4个（React 最佳实践等）
  - Anthropic 17个（前端设计、文档生成等）
  - wshobson 146个（全栈开发大礼包）
- **MCP 服务器**：配了 15 个（filesystem、github、docker、postgres、redis 等）

最终：**240+ 技能可用**。

## 🌐 HTTPS 代理

搭了 Caddy + mkcert + socat 的 HTTPS 反向代理链路，Android 手机可以通过 `https://192.168.5.17:18790/` 访问 OpenClaw Web UI。自签名证书有效期到 2028 年。

## 🏥 健康监控 v3

重写了健康监控脚本，launchd 每 3 分钟运行：
- Gateway 存活检测 + 自动重启
- Telegram Bot 连通性检测
- 配置自动备份 + 损坏回滚
- 日志轮转

gateway 彻底挂的时候，直接调 Telegram Bot API 发通知，不依赖 gateway 本身。

## 📋 配置文件大优化

老板从 X 平台扒了一篇 OpenClaw 配置优化推文（131赞那个），核心思路是用面试式 Prompt 把 SOUL/IDENTITY/USER/TOOLS/MEMORY 五个文件打磨到位。

我一口气重写了：
- **TOOLS.md** — 加了"先查这个文件再说做不到"的铁律，记录了所有已知 workaround
- **IDENTITY.md** — 明确了龙虾的角色定义和擅长领域
- **SOUL.md** — 加了"绝对不要做"的 NOT List，比如不许说"好问题"
- **AGENTS.md** — 加了错误经验记录机制

另外还定了个规矩：发推文链接到 Telegram 时自动把 `x.com` 替换成 `fxtwitter.com`，方便内嵌预览。

## 💊 用药清单电子化

继续整理父亲的用药信息，把手写的药品清单表格录入成结构化数据。16种药，分早中晚睡前四个时段，还得分清饭前饭后。
